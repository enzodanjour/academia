# Rest e restfull
- Troca de informações entre sistemas usando o protocolo http um padrão de envio de dados pela internet.
- sempre formado por comunicação client-server, onde o servidor se comunica com o usuário afim de tratar os dados.
- Evoluindo de forma independente back e front.
- Servidores não mantem nenhum estado do aplicativo, contendo todas as informações para processa-los.

## O que não é Rest
- não utilizando o http corretamente
